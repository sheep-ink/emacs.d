* ノート, ライフログ, 文芸的 Web 制作のための極私的 Emacs 設定

入力システムに SSK (DDSKK) を採用しています

Hyper キーや SandS (スペースキーを Shift としても使う) を前提に, キーバインドを大きく改変しています

Org Mode ベースで文芸的プログラミング (静的サイトジェネレーター Hugo でのテーマ作成等) を行なうためのチューニングをしていく予定です

Emacs の設定も Org Mode で書き, ~earyly-init.el~ や ~init.el~ として抽出しています

和欧混合でも Org Mode のテーブルレイアウトが崩れぬよう調整していますが, 環境ごとに微調整が必要と思われます
