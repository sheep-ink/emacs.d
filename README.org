* ノート·ライフログ·文芸的Web制作のための極私的Emacs設定

入力システムに SSK (DDSKK) を採用しています

Org Mode ベースで文芸的プログラミング (静的サイトジェネレーター Hugo でのテーマ作成等) を行なうためのチューニングをしています

Emacs の設定も Org Mode で書き, ~earyly-init.el~ や ~init.el~ として抽出しています

和欧混合でも Org Mode のテーブルレイアウトが崩れぬよう調整していますが, 環境ごとに微調整が必要と思われます

[[file:.github/assets/org-mode-preview-01.png]]

** Environment

- *OS* : Ubuntu 24.04 LTS
- *Emacs* : GNU Emacs 30.2 (PGTK build)
- *Input Method* : DDSKK
